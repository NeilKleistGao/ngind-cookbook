\doxysubsection{Detailed Description}
Basic class of all rendering component. This class is a abstract class so you can\textquotesingle{}t create an instance unless you implement the draw function. If your component will not show anything on the screen, use simple \mbox{\hyperlink{classngind_1_1components_1_1Component}{Component}} class instead.
